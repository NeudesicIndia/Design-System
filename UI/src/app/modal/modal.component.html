Modals

Neu Modal component is a simple lightweight re-usable web component for managing modals inside different framework web based projects. It was built for modern browsers using JavaScript, HTML5 and SCSS.

How to use it:

Check the below details to know how to use the modal box.

Setup

To use neu-modal-component in your project install it via npm:

npm i neu-modal
OR 
https://unpkg.com/neu-modal@1.0.4/dist/modal-component/modal-component.esm.js

Tag

Our Neu modal component has its own custom tag element:

  • Following is the Tag structure: 
     
<neu-modal size="md"> 
    
   <div slot="header"> ... </div> 
    
   <div slot="body"> ... </div> 
    
   <span slot="footer"> ... </div> 
    
</neu-modal>

Properties

Property values can be specified using corresponding attributes defined on the Neu Modal web component element.:

  • Following is the Tag structure: 
     
id = "anyIdName"  //ID is ref to handle events
    
size = "md"  //To define the size of your modal (small-sm, medium-md, large-lg)
    
showcancel = "true"     //To show or hide Cancel Button in footer - default true
    
showcancelicon = "false"  //To show or hide Cancel Icon in header - default true
    
footer = "true" //To show or hide modal footer - default true
    
header = "true" //To show or hide modal header - default true

Methods

Neu Modal has two methods to handle the events:

  • Following are the methods to interact with Neu Modal: 
     
openModal()
     
onModalClose()
  • To open a Modal - trigger openModal() method using any event 
    document.querySelector('refName').openModal(); //Native JS
